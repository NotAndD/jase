Roll a random character from the Poll:

To get general examples of how the character in this role should be.. or just to play with pre-made ready-to-play descriptions, roll a random character.

The name of the character. It will be used in the story wherever needed. Example: "[Caleb] lifted his hands and grinned"

The gender is used to select the correct words where needed (he or she? Him or her? Boy or Girl? And so on..) and it may be used to lock / unlock certain actions which can't work with both genders.

The full description will be used to describe the character only once, when the character first make his / her appearance.. so it isn't that important.

Just keep it short and also, it should work if used this way: "A [your full description here]."

Short descriptions are used in place of the name, wherever needed in the story to select the right subject. Insert multiple short descriptions, each one on a different line (the more, the better).

Examples: "The [muscled dragon] shifted position and grinned.", "Unfortunately for the [deer], her plan didn't work", "The [black feline] automatically made a step back, terrorized."

Offensive short descriptions works exactly as normal short descriptions. The main difference is that they are used when the subject needs to be sort-of humiliated or offended. Insert multiple short descriptions, each one on a different line (the more, the better).

Examples: "Sooo.. how are ya doing, [runt]?", "It was true.. compared with the others, she could be considered [weakling]".
