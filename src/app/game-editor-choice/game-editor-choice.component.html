Do you want to create or edit a Story?

(the process is slightly different)

Take into consideration a few things:
  • The Story ID must be unique across all stories (even the one you didn't create)
  • It may contains only letters, numbers and the - character.
  • Once the Story is published, people will be able to open it by knowing the Story ID, either by searching it or directly with the URL.
Please notice that:
  • Once the story is created, whoever has the Author ID will be able to UPDATE or DELETE it.
  • If you lose it, you will NOT be able to edit your story anymore and there's NO WAY to obtain it again.
  • If you still haven't understood, COPY this Author ID and store it somewhere safe.
